Value set CareConnect-ConditionClinicalStatus-1

The official URL for this value set is: 

http://hl7.org.uk/CareConnect-ConditionClinicalStatus-1.valueset.xml

Version

1.0

Description

Condition Clinical Status codes.

Definition

This valueset is comprised of the following codes from inline code system http://hl7.org.uk/fhir/ValueSet/CareConnect-ConditionClinicalStatus:

CodeDisplayDefinition
activeActiveThe subject is currently experiencing the symptoms of the condition.
inactiveInactiveThe subject is either in remission or resolved.
relapseRelapseThe subject is re-experiencing the symptoms of the condition after a period of remission or presumed resolution.
remissionRemissionThe subject is no longer experiencing the symptoms of the condition, but there is a risk of the symptoms returning.
resolvedResolvedThe subject is no longer experiencing the symptoms of the condition and there is no perceived risk of the symptoms returning

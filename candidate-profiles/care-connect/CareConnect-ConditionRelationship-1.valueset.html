Value set CareConnect-ConditionRelationship-1

The official URL for this value set is: 

http://hl7.org.uk/CareConnect-ConditionRelationship-1.valueset.xml

Version

1.0

Description

A set of codes specifying the kind of relationship that exists with a target condition.

Definition

This valueset is comprised of the following codes from inline code system http://hl7.org.uk/fhir/ValueSet/CareConnect-ConditionRelationship:

CodeDisplayDefinition
combinedCombinedThis condition is combined/merged with the target condition to form a single logical condition.
groupedGroupedThis condition is part of a group that includes the target condition as a member of the group.
replacesReplacesThis condition replaces a previous condition (i.e. a revised diagnosis). The target condition is now obsolete.
evolved-fromEvolved FromThis condition has evolved from the target condition.
onsetOnsetThis condition follows-on from the target onset condition. Used when specifing a relationship to a previous onset condition when this condition episodicity is Review, Flare or End.

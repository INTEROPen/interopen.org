Value set CareConnect-EthnicCategory-1-Valueset

The official URL for this value set is: 

http://hl7.org.uk/CareConnect-EthnicCategory-1-VS

Description

The vocabulary describes the ethnic category which the patient regards as their own, designed to be used with later versions of PDS. It is an extension of the ETHNIC CATEGORY CODE described in the NHS Data Model and Dictionary for England.

Referencehttp://www.datadictionary.nhs.uk/data_dictionary/data_field_notes/p/pds/pds_ethnic_category_code_de.asp?shownav=0
OIDurn:oid:2.16.840.1.113883.2.1.3.2.4.16.6

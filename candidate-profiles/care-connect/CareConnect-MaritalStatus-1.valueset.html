Value set CareConnect-MaritalStatus-1

The official URL for this value set is: 

http://hl7.org.uk/CareConnect-MaritalStatus-1.valueset.xml

Version

1.0

Description

This value set defines the set of codes that can be used to indicate the marital status of a person.

OIDurn:oid:2.16.840.1.113883.4.642.2.19

Requirements

The codes listed in this FHIR ValueSet are a restricted set of the FHIR Marital Status set of codes. This FHIR ValueSet has a 'Required' binding; for conformance the Marital Status element SHALL include a code listed below. Concept Maps for this value set: http://hl7.org.uk/CareConnect-MaritalStatus-DDMAP-1.valueset.xml

Definition

This valueset is comprised of the following codes from inline code system http://hl7.org/fhir/marital-status:

CodeDisplayDefinition
UUnmarriedThe person is not presently married. The marital history is not known or stated.

AND the following codes from external code system http://hl7.org.uk/CareConnect-MaritalStatus:

CodeDisplay
DDivorced
LLegally Separated
MMarried
SNever Married
WWidowed

AND the following codes from external code system http://hl7.org/fhir/v3/NullFlavor:

CodeDisplay
UNKunknown

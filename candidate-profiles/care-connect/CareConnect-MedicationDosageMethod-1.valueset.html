Value set CareConnect-MedicationDosageMethod-1

The official URL for this value set is: 

http://hl7.org.uk/CareConnect-MedicationDosageMethod-1.valueset.xml

Version

1.0

Description

A code from the SNOMED Clinical Terminology UK coding system that describes the e-Prescribing method. Any code from the SNOMED CT UK 'NHS e-Prescribing method subset' with subset original id 30101000001130; the corresponding SNOMED CT UK Refset fully specified name is 'ePrescribing method simple reference set (foundation metadata concept)' with Refset Id 999000041000001103.

Definition

This valueset is comprised of codes from external code system http://snomed.info/sct where:

  • concept in 999000041000001103

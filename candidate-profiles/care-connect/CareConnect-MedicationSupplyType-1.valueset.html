Value set CareConnect-MedicationSupplyType-1

The official URL for this value set is: 

http://hl7.org.uk/CareConnect-MedicationSupplyType-1.valueset.xml

Version

1.0

Description

Indicates the legal/contractual nature of a prescription. This valueset references the vocabulary for EhrSupplyType in the NPfIT Message Implementation Manual 7.2.02.

Definition

This valueset is comprised of the following codes from inline code system http://snomed.info/sct:

CodeDisplay
394823007NHS Prescription
394824001Private Prescription
394825000ACBS Prescription
394826004OTC sale
394827008Personal Administration

Resource CareConnect-Condition-1

The official URL for this profile is: 

http://hl7.org.uk/CareConnect-Condition-1

Convergence status: Endeavour-only draft.

Description

Use to record detailed information about conditions or diagnoses recognised by a clinician. The Condition profile specifically excludes Problems as those are handled with their own profile.

Definition

This profile is derived from Condition.

Extension CareConnect-ConditionProblemEpisode-1-Extension

The official URL for this profile is: 

http://hl7.org.uk/CareConnect-ConditionProblemEpisode-1-Ext

Convergence status: NHS Digital-Endeavour convergence profile.

Description

The problem episode of which this condition is part of. The episodicity type (Review, Flare, End) indicates the type of relationship that exists between this follow-on condition and the target problem episode.

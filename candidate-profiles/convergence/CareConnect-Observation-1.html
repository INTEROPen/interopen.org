Resource CareConnect-Observation-1

The official URL for this profile is: 

http://hl7.org.uk/CareConnect-Observation-1

Convergence status: NHS Digital-Endeavour convergence profile.

Description

The Observation resource is used for tracking the current and historical observations that have been made for a patient.

Definition

This profile is derived from Observation.

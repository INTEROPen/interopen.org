The official URL for this profile is: 

http://hl7.org.uk/CareConnect-Problem-Condition-1

Convergence status: NHS Digital-Endeavour convergence profile.

Description

Problem Condition resource is used to record detailed information about problems recognised by a clinician.

This profile is derived from Condition.
